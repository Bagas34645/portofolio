Python adalah bahasa pemrograman high-level yang dikenal dengan sintaks bersih, ekosistem library raksasa, dan versatilitas. ## Mengapa Python?

- **Readable**: Indentasi sebagai blok kode, dipaksa menulis rapi
- **Batteries included**: Standard library kaya (os, json, http, sqlite, dll.)
- **Ekosistem**: PyPI menyediakan 500.000+ package
- **Versatile**: Web (Django/FastAPI), Data (Pandas/NumPy), ML (PyTorch/scikit-learn), Automation
- **Demand tinggi**: Salah satu bahasa paling dicari di industri

## Sintaks Python Dasar

Python case-sensitive dan tidak membutuhkan titik koma di akhir baris (opsional):

```python
# Satu statement per baris (umum)
print("Hello")
print("World")

# Beberapa statement dalam satu baris (hindari kecuali sangat singkat)
x = 1; y = 2

# Statement panjang bisa dipecah dengan backslash atau tanda kurung
total = (1 + 2 + 3 +
         4 + 5)

if True:
    print("Blok if")   # indentasi wajib
```

Aturan penting:

- **Indentasi 4 spasi** menandai blok kode (if, for, def, class)
- **Nama variabel**: huruf, angka, underscore; tidak boleh diawali angka
- **Kata kunci** (`if`, `for`, `class`, dll.) tidak boleh dipakai sebagai nama variabel

## Output dengan print()

`print()` menampilkan teks atau nilai ke stdout:

```python
print("Hello, World!")           # teks
print(42)                         # angka
print("Pi =", 3.14159)            # beberapa argumen, dipisah spasi
print("a", "b", "c", sep="-")     # a-b-c
print("Loading", end="...")       # tanpa newline di akhir
print()                           # baris kosong

# f-string: format output paling direkomendasikan
name = "Alice"
print(f"Halo, {name}! Umur: {25}")
```

## Input dari User

```python
nama = input("Siapa nama kamu? ")
print(f"Halo, {nama}!")

# input() selalu mengembalikan string, konversi eksplisit jika perlu angka
umur = int(input("Berapa umur kamu? "))
print(f"Tahun depan umurmu {umur + 1}")
```

## REPL: Interactive Python

REPL (Read-Eval-Print Loop) sebuah playground instan untuk eksperimen:

```python
$ python3
Python 3.12.x (...)
>>> 2 + 2
4
>>> "hello" + " world"
'hello world'
>>> len("Python")
6
>>> type(42)
<class 'int'>
>>> exit()
```

Shortcut: `Ctrl+D` untuk keluar.

```

Jalankan:

```

## Struktur Program Python

```python
#!/usr/bin/env python3
"""Docstring modul: jelaskan apa yang dilakukan file ini."""

import sys


def main():
    """Entry point program."""
    print(f"Python {sys.version}")
    print("Program berjalan!")


if __name__ == "__main__":
    main()
```

Pattern `if __name__ == "__main__"` memastikan `main()` hanya dipanggil saat file dijalankan langsung, bukan saat di-import.

## Indentasi = Blok Kode

Python menggunakan indentasi (4 spasi standar) sebagai penanda blok, bukan kurung kurawal:

```python
if True:
    print("Ini di dalam blok if")
    print("Masih di dalam")
print("Ini di luar blok if")
```

Mixing tab dan spasi = error. Gunakan **4 spasi** konsisten (konfigurasi editor).
